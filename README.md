# Dynamic-Pathfinding-Agent
Python Tkinter grid pathfinding visualizer implementing A* and Greedy Best-First Search with Manhattan &amp; Euclidean heuristics. Interactive grid editor, real-time animation, maze generator, dynamic obstacle spawning with automatic re-planning, and live metrics (nodes visited, path cost, compute time).

To run this project, first clone the repository to your local machine and navigate into the project directory. Make sure you have Python 3.8 or higher installed on your system. You can verify your Python installation by running python --version in your terminal.

This project uses only built-in Python libraries such as tkinter, heapq, math, random, and time. No external dependencies are required. If you are using Linux and encounter issues with Tkinter, you may need to install it manually using your package manager (e.g., sudo apt install python3-tk).

Running the Application

After cloning the repository and ensuring Python is installed, run the application using:

python pathfinding_agent.py

or (depending on your system):

python3 pathfinding_agent.py

The graphical interface will open, allowing you to configure the grid, select the algorithm and heuristic, generate obstacles, and visualize the pathfinding process in real time.
